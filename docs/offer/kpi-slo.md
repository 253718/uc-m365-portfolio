## KPI / SLO (delivery & stabilization)

Purpose: define measurable signals that make delivery predictable.

### Delivery KPIs

- percentage of migrations using the full artefact set
- percentage of runs producing evidence packs
- percentage of cutovers with rollback triggers defined

### Stabilization examples

- time to acknowledge
- time to triage
- time to mitigate
- incident severity distribution during hypercare
